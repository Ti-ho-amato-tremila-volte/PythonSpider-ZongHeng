# ZonHengSpider
## 纵横中文网爬虫及数据可视化分析
 本项目基于Python Flask框架、Python爬虫、Echarts、WordCloud等技术实现对纵横中文网小说月票榜、点击榜、推荐榜、新书榜、24小时更新榜的数据可视化分析。
 ***
### *spider*文件目录下是项目爬虫部分，分别将获取的数据保存为Excel文件与SQLite文件
### *static*文件与*templates*文件下是数据可视化分析需要用到的一些模板与素材
